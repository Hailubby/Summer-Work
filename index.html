Welcome to COMPSYS 202/MECHENG 270


Compsys 202/Mecheng 207 is a project based course with an extensive hands-on programming experience. In this course, you will learn about object oriented programming and concepts through c++.


This website contains code examples and animations which aim to aid the student's understanding of how variable and objects are organised in memory.

There are examples on basic variables, references, pointers, constants as well as an introduction to classes and objects


A few things to note about playing the animations:

  • "Start" plays the animation from beginning to end.
  • "Step" plays the animation of the next line, allowing you to go through the code at your own pace.
  • Click on any line to see the animation for it.
