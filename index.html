Visualising basic programming concepts


This website contains code examples and animations which aim to aid understanding of how variable and objects are organised in memory, here particularly focusing on C/C++.

There are examples on basic variables, references, pointers, constants as well as an introduction to classes and objects


A few things to note about playing the animations:

  • "Start" plays the animation from beginning to end.
  • "Step" plays the animation of the next line, allowing you to go through the code at your own pace.
  • Click on any line to see the animation for it.
