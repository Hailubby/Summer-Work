Definitions

Declaration: a declaration states the type of a variable, along with its name. A variable may be only declared once and must be declared before reading or assigning.

Initialisation: the first assignment of a value to a variable. Initialisation of a variable may be done in conjunction with the declaration.

Definitions

Assignment: throwing away the old value of a variable and replacing it with a new one.

References

A reference binds a variable name to an existing variable acting as an alias. Once initialised, either the variable name or reference name may be used to refer to the variable

  • You cannot have NULL references
  • A reference must be initialised when created
  • Once a reference is initialised to an object, it cannot be changed to refer to another object

Pointers

A pointer is a variable whose value is the address of another variable.

  • Pointers must be declared before they can be used
  • It is good practise to initialise pointers either to an address (using &) or "null"
  • Use * to access the value at the address available in the pointer variable
  • Unlike references, a pointer can be copied and also later be bound to other variables
